# تطبيق رونق - Ronaq App

تطبيق ويب حديث وأنيق لطلب المشروبات الباردة والساخنة والحلويات بهوية بصرية فاخرة.

## المميزات

✨ **هوية بصرية راقية**
- ألوان عنابية داكنة وكريمية فاخرة
- شعار "رونق" مع أيقونة وردة
- خط Tajawal العربي الأنيق
- تصميم حديث مع حواف دائرية ناعمة

📱 **تصميم متوافق مع الجوال**
- Mobile-first design
- تجربة سلسة على جميع الأجهزة
- واجهة بديهية وسهلة الاستخدام

🛒 **نظام الطلبات والسلة**
- اختيار خيارات الطلب (استلام/توصيل/من السيارة)
- تصفح التصنيفات المختلفة
- إضافة المنتجات إلى السلة
- تعديل الكميات بسهولة
- عرض الإجمالي والدفع

👤 **إدارة الحساب الشخصي**
- عرض المحفظة والنقاط
- تتبع الطلبات السابقة
- نظام المكافآت والعروض
- خيارات الحساب المختلفة

## كيفية الاستخدام

### فتح التطبيق
1. استخرج ملفات التطبيق
2. افتح ملف `index.html` في متصفحك
3. استمتع بتجربة التسوق!

### الملاحة
- استخدم شريط التنقل السفلي للتنقل بين الصفحات
- اضغط على التصنيفات لعرض المنتجات
- استخدم أيقونة السلة لعرض طلباتك

### الطلب
1. اختر خيار الطلب (استلام/توصيل/من السيارة)
2. اختر التصنيف المطلوب
3. أضف المنتجات إلى السلة
4. عدّل الكميات حسب الحاجة
5. اضغط "تأكيد الطلب"
